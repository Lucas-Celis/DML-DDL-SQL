# DML-SQL
🔷 Estrutura: DDL x DML, qual a diferença entre eles?
Entender a definição de ambas e saber distinguir qual o papel de cada uma é essencial para a governança.
- DDL (Data Definition Language): define, modificanda ou exclui a estrutura do banco, utilizando dos comandos "CREATE", "ALTER" e "DROP", onde o primeiro é a criação de tabela, o segundo é alteração de uma tabela e o último a exclusão de uma tabela (Foco nas tabelas e colunas);
- DML (Data Manipulation Language): irá manipular os dados contidos nas estruturas, utilizando dos comandos de "INSERT", "UPDATE" e "DELETE", onde o primeiro insere dados, o segundo atualiza um dado e o terceiro deleta um dado (Foco nos dados e linhas da tabela).
